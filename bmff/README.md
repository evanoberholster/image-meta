# BMFF Package

## Description

Golang library for parsing ISOBMFF Image files specifically Heif/Heic, AV1/AVIF and CR3.

### Parsing File Type Support

- Heif/Heic (Partial)
- AV1/Avif (WIP)
- CR3 (WIP)
